课程介绍：

不论何种互联网产品，都难以逃离“商业化”的发展路径，这就意味着产品经理必须要懂得如何搭建电商系统。但实际上，大多数产品经理受限于自身和业务的局限，都没有亲自操盘过一个大型、完整、标准化的电商项目。因此，我把自己多年的实战经验和产品思考都融在这个经典电商项目中，希望你在学习之后，不仅掌握系统的业务方案，同时获得能力的进阶，最终成长为一个更有价值的高阶产品经理！
